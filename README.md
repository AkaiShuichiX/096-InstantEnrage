# 096 InstantEnrage - SCP:SL
Exiled SCP: SL Plugin that when you look at SCP-096 will instantly make SCP-096 angry

This plugin is only compatible with [Exiled Framework]([https://github.com/Exiled-Team/EXILED])

## Configuration
```yaml
old_096:
  is_enabled: true
  debug: false
  instant_rage: true
```
